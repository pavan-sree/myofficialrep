Automated Rain Detection and Protection System

This project leverages Arduino technology, incorporating a rainwater sensor, servo motor, and DC fan. The rain sensor promptly detects rainfall, triggering the servo motor to deploy a protective shield over vulnerable surfaces, safeguarding garments from inclement weather. This initiative marks my initial foray into Arduino projects, demonstrating successful implementation while mastering Arduino IDE and the platform itself.

Robotic Arm

The robotic arm utilizes servo motors controlled by Arduino, augmented with three potentiometers for precise manipulation. Designed to enhance industrial efficiency and reduce manpower, this model initially revealed inherent limitations during my research. Subsequently, recognizing the need for improvement, I decided to integrate hand gesture control. This innovation aims to overcome existing constraints, ensuring greater adaptability and operational effectiveness in industrial settings.

Gesto Mimicking Limb

This is modified robotic arm, which responds to hand gestures, powered by Arduino and employing an HC-05 module for Bluetooth communication. The integration facilitates real-time interaction between the robotic arm and external software developed using Streamlit with OpenCV, Mediapipe, and Bluetooth socket programming. Utilizing webcam input, the software captures and transmits hand gesture data via Bluetooth to the Arduino, which controls three servos to manipulate objects. This innovative project aims to enhance industrial efficiency and foster greater human-robot collaboration.

CRUD Website Using Angular with Firebase
